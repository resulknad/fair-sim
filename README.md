## Installation
1. Clone this repository
2. Install [pipenv](https://github.com/pypa/pipenv)
3. Run `pipenv install` 

## Basic Usage
1. Run `jupyter notebook`
2. Open `main_plots.py` to see how to parametrize, run and analyse a simulation
